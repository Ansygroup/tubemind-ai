# 💾 الخطوة 2: إعداد قاعدة البيانات

## Supabase Setup

1. روح: https://app.supabase.com
2. افتح مشروعك
3. SQL Editor → New Query
4. الصق محتوى ملف `database/schema.sql`
5. اضغط Run

✅ جداول المشروع:
- `channels` - معلومات القنوات
- `videos` - الفيديوهات المنتجة
- `jobs` - مهام الإنتاج
- `analytics` - إحصائيات الأداء
